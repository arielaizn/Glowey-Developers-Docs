# Conversations API

The Conversations API manages persistent session records for SuperGlow pipelines. Each conversation tracks metadata, generation state, and final outputs.

## GET /api/superglow/conversations

List the user's SuperGlow conversations (newest first, limit 50).

### Authentication

Bearer token required.

### Query Parameters

None. Returns the user's full conversation list.

### Response

```json
{
  "conversations": [
    {
      "id": "conv_19a8b7c6d5e4f",
      "slug": "quantum-explainer-001",
      "title": "Quantum Computing Explainer",
      "workflow": "video+music",
      "prompt": "Create an animated explainer video about quantum computing",
      "final_video_url": "https://storage.gloweyai.com/quantum_final.mp4",
      "created_at": "2025-05-19T10:30:00Z",
      "updated_at": "2025-05-19T10:45:30Z"
    }
  ]
}
```

| Field | Type | Description |
|-------|------|-------------|
| `id` | string (UUID) | Unique conversation ID |
| `slug` | string | URL-friendly identifier (auto-generated) |
| `title` | string | User-facing conversation title |
| `workflow` | string | Generation pipeline used (e.g., `video+music`, `image`) |
| `prompt` | string | Initial user prompt |
| `final_video_url` | string | URL of final generated video (if available) |
| `created_at` | ISO 8601 | Creation timestamp |
| `updated_at` | ISO 8601 | Last update timestamp |

### Errors

- `401` – Unauthorized
- `429` – Rate limited
- `500` – Server error

### Example Request

```bash
curl "https://gloweyai.com/api/superglow/conversations" \
  -H "Authorization: Bearer glow_sk_YOUR_TOKEN_HERE"
```

## POST /api/superglow/conversations

Create a new conversation record.

### Authentication

Bearer token required.

### Request Body

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `title` | string | Yes | Conversation title (max 200 chars) |
| `workflow` | string | No | Pipeline workflow identifier (e.g., `video+music`) |
| `prompt` | string | No | Initial user prompt or context |

### Response

```json
{
  "id": "conv_19a8b7c6d5e4f",
  "slug": "quantum-explainer-001",
  "title": "Quantum Computing Explainer",
  "workflow": null,
  "prompt": null,
  "created_at": "2025-05-19T10:30:00Z",
  "updated_at": "2025-05-19T10:30:00Z"
}
```

Returns the created conversation record (status 201).

### Errors

- `400` – Missing or empty `title`
- `401` – Unauthorized
- `429` – Rate limited
- `500` – Server error

### Example Request

```bash
curl -X POST https://gloweyai.com/api/superglow/conversations \
  -H "Authorization: Bearer glow_sk_YOUR_TOKEN_HERE" \
  -H "Content-Type: application/json" \
  -d '{
    "title": "Quantum Computing Explainer",
    "workflow": "video+music",
    "prompt": "Create an animated explainer about quantum superposition"
  }'
```

## GET /api/superglow/conversations/[id]

Retrieve a specific conversation by UUID or slug.

### Authentication

Bearer token required.

### URL Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `id` | string | UUID or slug of the conversation |

### Response

```json
{
  "id": "conv_19a8b7c6d5e4f",
  "slug": "quantum-explainer-001",
  "title": "Quantum Computing Explainer",
  "workflow": "video+music",
  "prompt": "Create an animated explainer video about quantum computing",
  "script_text": "Meet Quantum. Unlike classical bits...",
  "clips": [
    {
      "id": "clip_001",
      "imageUrl": "https://storage.gloweyai.com/clip_001.jpg",
      "duration": 3
    }
  ],
  "media": [
    {
      "type": "image",
      "url": "https://storage.gloweyai.com/media_001.jpg"
    }
  ],
  "music_url": "https://storage.gloweyai.com/music_background.mp3",
  "final_video_url": "https://storage.gloweyai.com/quantum_final.mp4",
  "steps": ["generate_images", "generate_video", "generate_music"],
  "created_at": "2025-05-19T10:30:00Z",
  "updated_at": "2025-05-19T10:45:30Z"
}
```

Extended fields returned on GET:

| Field | Type | Description |
|-------|------|-------------|
| `script_text` | string | Generated script or narration |
| `clips` | array | Video clip segments |
| `media` | array | Generated media assets |
| `music_url` | string | Background music URL |
| `final_video_url` | string | Final composite video URL |
| `steps` | array | Generation pipeline steps completed |

### Errors

- `401` – Unauthorized
- `404` – Conversation not found
- `429` – Rate limited
- `500` – Server error

### Example Request

```bash
curl "https://gloweyai.com/api/superglow/conversations/conv_19a8b7c6d5e4f" \
  -H "Authorization: Bearer glow_sk_YOUR_TOKEN_HERE"
```

## PATCH /api/superglow/conversations/[id]

Update conversation metadata and generation state.

### Authentication

Bearer token required.

### URL Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `id` | string | UUID or slug |

### Request Body

Fields allowed for update (all optional):

```json
{
  "title": "Updated Title",
  "script_text": "New script content...",
  "clips": [...],
  "media": [...],
  "music_url": "https://...",
  "final_video_url": "https://...",
  "steps": ["generate_images", "generate_video"],
  "workflow": "video+music",
  "prompt": "Updated prompt..."
}
```

### Response

```json
{
  "id": "conv_19a8b7c6d5e4f",
  "slug": "quantum-explainer-001",
  "title": "Updated Title",
  "updated_at": "2025-05-19T11:00:00Z"
}
```

### Errors

- `401` – Unauthorized
- `404` – Conversation not found
- `500` – Server error

### Example Request

```bash
curl -X PATCH https://gloweyai.com/api/superglow/conversations/conv_19a8b7c6d5e4f \
  -H "Authorization: Bearer glow_sk_YOUR_TOKEN_HERE" \
  -H "Content-Type: application/json" \
  -d '{
    "title": "Final Quantum Explainer",
    "final_video_url": "https://storage.gloweyai.com/quantum_final.mp4"
  }'
```

## DELETE /api/superglow/conversations/[id]

Delete a conversation (all associated data is removed).

### Authentication

Bearer token required.

### URL Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `id` | string | UUID or slug |

### Response

```json
{
  "ok": true
}
```

### Errors

- `401` – Unauthorized
- `404` – Conversation not found
- `500` – Server error

### Example Request

```bash
curl -X DELETE https://gloweyai.com/api/superglow/conversations/conv_19a8b7c6d5e4f \
  -H "Authorization: Bearer glow_sk_YOUR_TOKEN_HERE"
```

## POST /api/superglow/conversations/[id]/beacon

Special endpoint for `navigator.sendBeacon` (page unload). Accepts only POST and persists conversation state without requiring custom headers.

**Note:** This endpoint is for internal browser use and tracking page exits. Most API integrations do not need it.

### Authentication

Bearer token sent as request body (fallback when headers unavailable).

### Request Body

Same as PATCH — any conversation fields to update.

### Response

```json
{
  "ok": true
}
```

### Errors

- `400` – Invalid JSON
- `401` – Unauthorized
- `500` – Server error

### Example (Browser JavaScript)

```javascript
navigator.sendBeacon(
  `https://gloweyai.com/api/superglow/conversations/conv_19a8b7c6d5e4f/beacon`,
  JSON.stringify({
    title: "Unsaved Work",
    script_text: "Current script state...",
    steps: ["generate_images"]
  })
);
```

## Conversation State Lifecycle

1. **Create** — `POST /conversations` → new conversation record
2. **Populate** — Submit prompt to Hermes, which generates and updates the conversation
3. **Monitor** — Poll conversation GET endpoint to fetch latest state
4. **Refine** — Send follow-up messages to Hermes with the same `conversation_id`
5. **Export** — Retrieve final assets from `final_video_url` or `media[]`
6. **Cleanup** — `DELETE /conversations/[id]` when done
